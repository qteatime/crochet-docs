Commands¶

If Types provide the structure for data that a Crochet program manipulates—or “what data looks like”—, then Commands describe how they behave. Commands are one of central concepts of Crochet, and in this section we’ll dive deep into what they are, how they work, and other things you should keep in mind when using them.
